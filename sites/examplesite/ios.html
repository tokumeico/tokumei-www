How to access Example Site

  1. Install Onion Browser

    2. In order to protect anonymity and free speech, Example Site is only accessible using Tor. Learn more.

    This guide will help you get Tor set up through the iOS port Onion Browser.

    Open the App Store and search for Onion Browser. Install the app published by Mike Tigas.

  2. Navigate to Example Site

    3. Just open Onion Browser and enter http://example.onion/ in its address bar.


    You can bookmark Example Site to come back without needing to remember the address.

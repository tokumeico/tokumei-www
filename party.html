Join us November 3rd in celebrating Tokumei's first birthday and the release of version 1.0.

Tokumei 1.0?

Tokumei's first real release was made available at midnight EDT today.

The 1.0 release includes a complete, high quality implementation of every feature we had initially planned for Tokumei. We will continue to actively develop Tokumei with fixes, improvements, and extra features; but the basic feature set is complete. Tokumei should now be stable, fast, and flexible; and we will fully support every release as of 1.0.

Of course, Tokumei's ISC source code always has been and always will be available for free via git. But 1.0 marks the first version of Tokumei released as a PGP-signed archive you can download over HTTP.
